# TicTacToe
This is my custom batch (command prompt) script for the original TicTacToe game.

# Progress
It all started when I saw a video on youtube creating backend API servers for games. I liked the idea so I started working on a **Node.Js backend** server, without any goal with it. Then a few days later, **Loftcreator**, one of my best friends came to me and asked, if I could teach them some of the basics in batch. So we started by creating our projects folder and worked on some stuff like the **Main menu**, and the **Language selection**. But then I thought about the backend I made: *"Why do I need a backend if I don't even have a frontend?"* So I decided to work on a simple offline TicTacToe, which Loftcreator and I made, as well as a **"Kill the Bot"** game mode, where you play against a pretty clever bot, **Loftcreator mainly coded**.
